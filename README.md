# Chakai Theme

A tea ceremony inspired theme for Micro.blog featuring hero image cards on the index page.

## Features

- **Hero Image Cards**: Index page displays posts as visual cards with hero images
- **Wabi-sabi Aesthetics**: Minimalist design inspired by Japanese tea ceremony
- **Japanese Typography**: Optimized for both Japanese and English text
- **Responsive Grid**: Cards automatically adjust to screen size
- **Lazy Loading**: Images load efficiently as you scroll

## Installation

1. Go to your Micro.blog account settings
2. Navigate to **Design** > **Custom Themes**
3. Click **New Theme**
4. Enter the GitHub URL: `https://github.com/Joi/chakai-theme`
5. Click **Clone**
6. Set as your active theme

## Customization

The theme uses CSS variables for easy customization. Edit `static/css/chakai.css` to modify:

- Colors (washi paper, sumi ink inspired)
- Typography (Hiragino Mincho, Noto Serif JP)
- Spacing and layout
